## Test Set Complet

### Catégorie 1 : Résumé (3)

1. "Résume-moi le dossier du client ABC"
2. "Donne-moi une vue d'ensemble du client XYZ"
3. "Quelles sont les informations importantes sur ce client ?"

### Catégorie 2 : Recherche Document (3)

4. "Quel est le dernier contrat pour ce client ?"

6. "Liste tous les documents du dossier"

### Catégorie 3 : Recherche Information (3)

7. "Quelle est la durée du contrat ?"
8. "Y a-t-il une clause de résiliation ?"
9. "Quel est le montant du contrat ?"

### Catégorie 4 : Préparation RDV (3)

10. "Prépare-moi une synthèse avant mon rendez-vous"
11. "Prépare-moi un briefing avant la réunion"
12. "Que dois-je savoir avant de rencontrer ce client ?"

### Catégorie 5 : Génération Mail (2)

13. "Rédige-moi un mail de suivi"
14. "Prépare un mail pour le renouvellement"

### Catégorie 6 : Absence Info (3)

15. "Quel est le chiffre d'affaires avec ce client ?"
16. "Qui est le décideur chez ce client ?"
17. "Quelle est la marge réalisée ?"

### Catégorie 7 : Client Ambigu (2)

18. "Résume le dossier Alpha"
19. "Infos sur client Test"

### Catégorie 8 : Permissions (2)

20. [Sans accès] "Dossier ABC"
21. [Accès partiel] "Dossier XYZ"

### Catégorie 9 : Non-Régression (2)

22. "Quelle est la météo demain ?" (hors sujet)
23. "Connecte-toi au CRM" (hors périmètre)

### Catégorie 10 : Sources (1)

24. "Cites-moi les sources utilisées"
