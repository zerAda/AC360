# Exemples Utilisateurs — Phase Complémentaire

## Vue d'Ensemble

Ce document contient des exemples de questions que les commerciaux peuvent poser à l'agent.

---

## Résumé de Dossier

| Question | Description |
|----------|-------------|
| "Résume-moi le dossier du client ABC" | Vue d'ensemble du client |
| "Donne-moi une synthèse du client XYZ" | Points clés |
| "Quelles sont les infos importantes sur ce client ?" | Points préoccupants |

## Questions Hors Périmètre

L'agent doit décliner ces questions :

| Question | Reason |
|----------|--------|
| "Quel chiffre d'affaires ?" | Donnée financière non disponible |
| "Qui est le décideur ?" | Information non dans SharePoint |
