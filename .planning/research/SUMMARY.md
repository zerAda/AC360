# SUMMARY.md — Copilot Studio + SharePoint Ecosystem Research

## Key Findings

### Stack

- **Required**: Microsoft 365, Copilot Studio, SharePoint, Teams, Entra ID
- **Optional**: Microsoft 365 Copilot (recommended for better search)
- **Out of scope**: Azure AI Search, Dataverse, Purview

### Table Stakes

- Natural language query understanding
- Folder summarization
- Document search
- Permission enforcement
- "Not found" responses
- Source citation

### Differentiators

- Meeting preparation
- Risk identification
- Email drafting
- Clause extraction

### Watch Out For

- Permission configuration errors
- SharePoint search indexing delays
- Vague question handling
- Large file processing (>7MB)
- Ambiguous client name resolution
