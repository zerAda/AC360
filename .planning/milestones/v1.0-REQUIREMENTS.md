## Requirement Outcomes

All v1 requirements were validated during the milestone audit and subsequent gap-fix session:

- **FILT-01** required an additional normalization step to handle accented characters ("Récap" → "RECAP") before filtering. Fixed in `core.py:Filters.filter_excel()`.
- **CALC-05** required an explicit cross-check between the sum of individual gaps and the expected total (`total_excel - total_pdf`). Fixed in `core.py:EcartCalculator.calculer_ecarts()`.
- **PERF-01** required a runtime benchmark. Fixed in `main.py:run_audit()` with `time.time()` and console alert if > 30s.
- **PERF-03** required explicit memory cleanup. Fixed in `main.py:run_audit()` with `gc.collect()`.
- **POST-01/POST-02** required secure file deletion. Fixed in `main.py:_safe_delete_file()` which only deletes files within the project directory.
