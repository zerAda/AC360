| T-03-09 | Information Disclosure | Teams webhook URL exposure | mitigate | webhookReceivers URL supplied as a parameter/secret (not committed); operator provisions a Power Automate/Workflows webhook (legacy O365 connectors retiring) — checkpoint Task 4 |
| T-03-10 | Information Disclosure | alert payloads carrying PII | mitigate | Alerts query aggregate counts/metrics + redacted App Insights data (RedactingSpanProcessor upstream, Plan 01/02); useCommonAlertSchema true; no raw PII in alert criteria |
| T-03-11 | Tampering | App Insights = classic (deprecated) | mitigate | Workspace-based component (WorkspaceResourceId set); classic is retired (RESEARCH State of the Art) |
| T-03-12 | Information Disclosure | EU data residency for telemetry (RGP) | mitigate | LAW retentionInDays 30 (RGP-04 short retention); App Insights region follows location param (EU); webtest Locations EU-only (RGP-06) |
