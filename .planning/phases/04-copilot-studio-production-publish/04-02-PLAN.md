<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| signed-in Teams user (1:1) → Copilot Studio agent | Silent SSO token-exchange must yield System.User.AccessToken for the prod API audience; wrong auth mode breaks it. |
| Copilot Studio agent → prod gateway | The agent forwards the user Bearer token to `ac360-gateway-prod`; audience/endpoint must match the Phase 2 app reg. |
| install scope (1:1 vs channel/group) | OBO + SharePoint RAG and manual-auth SSO are only honored in 1:1; channel/group exposure is a cross-user disclosure boundary. |
| document content → LLM | Document text is untrusted content that may carry injection/exfiltration instructions. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-06 | Elevation of Privilege | wrong auth mode → no user token → non-RBAC fallback | mitigate | Runbook hard-codes "Authenticate manually" Entra V2 (LOCKED); rejects "Authenticate with Microsoft"; gateway AC360_REQUIRE_OBO (Phase 1) refuses non-OBO; happy-path acceptance check (Task 3 item 6). |
| T-04-07 | Information Disclosure | channel/group install exposes cross-user SharePoint data | mitigate | 1:1-only enforcement (PUB-05): "Allow add to team" OFF; rationale documented; Task 3 item 4 verifies scope. |
| T-04-08 | Spoofing | token forwarded to wrong audience | mitigate | Scopes/token-exchange URL bound to the prod API audience; gateway validates `aud` (Phase 1 auth.py); audience scope resolved from Phase 2 outputs, never hardcoded in repo. |
| T-04-09 | Tampering / Information Disclosure | prompt injection / system-prompt exfiltration via document content | mitigate | `contentModeration: High` (platform RAI input+output filter) + agent.mcs.yml anti-injection rules; live known-blocked-prompt test (Task 3 item 5) → GUARDRAILS_VALIDATION.md evidence. |
| T-04-10 | Information Disclosure (residency) | non-EU token service / data egress | mitigate | EU redirect `europe.token.botframework.com` + EU env confirmation (PUB-01/RGP-06); Task 3 items 1-2. |
| T-04-SC | Tampering | npm/pip/cargo installs | mitigate | No package installs in this plan (docs only); optional `markdownlint-cli` is dev-time lint, not a runtime dependency and not added to requirements.txt. |
</threat_model>
