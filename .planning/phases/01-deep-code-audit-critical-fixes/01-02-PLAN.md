| app memory -> logs/telemetry | Dynamic strings and dimension dicts cross into observable sinks; must be redacted (AUD-06). |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-01-02-02 | Information disclosure | telemetry dimensions / detail strings | mitigate | New dict-value redaction helper routes every dimension value through the single audited redact() surface (wired at call sites in 01-04/01-06). |
