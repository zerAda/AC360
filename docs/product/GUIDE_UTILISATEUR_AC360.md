> **Version :** 2.1.0 | **Date :** Juin 2026 | **Public :** Équipes commerciales GEREP

## 🎯 Qu'est-ce qu'AC360 ?

**AC360** est votre assistant IA intégré à Microsoft Teams. Il vous permet d'interroger instantanément vos dossiers clients SharePoint en langage naturel, sans quitter votre environnement de travail.

> 🔒 **Important :** AC360 ne consulte que les documents auxquels vous avez personnellement accès sur SharePoint GEREP. Il ne modifie jamais vos documents.

---

## 🚀 Démarrer avec AC360

### Comment y accéder
1. Ouvrez **Microsoft Teams**
2. Cliquez sur **Chat** dans le menu gauche
3. Recherchez **AC360** dans vos contacts
4. Démarrez une conversation — AC360 vous accueille automatiquement

### Premier message
Dès que vous ouvrez la conversation, AC360 vous propose un menu de démarrage rapide avec des suggestions de questions. Cliquez simplement sur l'une d'elles ou tapez votre propre question.

## 📋 Les 11 Capacités d'AC360

### 1. 📊 Résumé Dossier Client
**Ce que ça fait :** Génère une synthèse commerciale complète d'un client en quelques secondes.

**Exemples de questions :**
```
Résume-moi le dossier du client [Nom]
Fais-moi une synthèse de [Nom]
Que sait-on sur le client [Nom] ?
```

**Ce que vous obtenez :** Tableau d'informations clés, points d'attention, prochaines actions, sources citées.

---

### 2. 📅 Préparation RDV Renouvellement
**Ce que ça fait :** Génère une fiche de préparation stratégique avant un rendez-vous client.

**Exemples de questions :**
```
Prépare mon RDV de renouvellement avec [Nom]
Aide-moi pour mon rendez-vous client [Nom]
Fais-moi une fiche de préparation pour [Nom]
```

**Ce que vous obtenez :** Situation actuelle (SWOT), enjeux financiers, stratégie conseillée, points à aborder.

---

### 3. ⚠️ Points d'Attention Client
**Ce que ça fait :** Identifie les risques, anomalies et alertes sur un dossier client.

**Exemples de questions :**
```
Quels sont les risques sur le dossier [Nom] ?
Points d'attention pour [Nom]
Y a-t-il des alertes sur ce client ?
```

---

### 4. 📄 Documents Manquants
**Ce que ça fait :** Vérifie la complétude du dossier et liste les documents à obtenir.

**Exemples de questions :**
```
Quels documents manquent pour [Nom] ?
Le dossier [Nom] est-il complet ?
Que me faut-il obtenir pour [Nom] ?
```

---

### 5. 🎤 Arguments de Vente
**Ce que ça fait :** Génère des arguments commerciaux personnalisés pour un client.

**Exemples de questions :**
```
Donne-moi les arguments de vente pour [Nom]
Comment convaincre [Nom] de renouveler ?
Arguments cross-selling pour [Nom]
```

---

### 6. ✉️ Brouillon Mail Commercial
**Ce que ça fait :** Rédige un email professionnel personnalisé à partir des données client.

**Exemples de questions :**
```
Rédige un mail de suivi pour [Nom]
Prépare un email de relance pour [Nom]
Écris un mail de confirmation de RDV pour [Nom]
```

---

### 7. 🏆 Tableau de Bord Portefeuille
**Ce que ça fait :** Vue macro de l'ensemble de votre portefeuille clients.

**Exemples de questions :**
```
Vue d'ensemble de mon portefeuille
Tableau de bord commercial
Quels sont mes clients actifs ?
```

**Ce que vous obtenez :** Liste clients, renouvellements urgents, alertes actives, indicateurs globaux.

---

### 8. ⏰ Alertes Renouvellement
**Ce que ça fait :** Identifie les contrats arrivant à échéance et priorise les actions.

**Exemples de questions :**
```
Quels contrats expirent dans 30 jours ?
Alertes renouvellement 60 jours
Contrats à renouveler ce trimestre
```

**Ce que vous obtenez :** Liste triée par urgence, actions prioritaires recommandées.

---

### 9. 📉 Suivi Sinistralité
**Ce que ça fait :** Analyse l'historique des sinistres et le ratio S/P d'un client.

**Exemples de questions :**
```
Quelle est la sinistralité de [Nom] ?
Historique sinistres de [Nom]
Résultat technique du client [Nom]
```

**Ce que vous obtenez :** Bilan S/P par année, sinistres significatifs, tendance, impact tarifaire probable.

---

### 10. 🔍 Analyse Gap de Couverture
**Ce que ça fait :** Identifie les garanties absentes ou insuffisantes — opportunités de cross-selling.

**Exemples de questions :**
```
Le client [Nom] est-il bien couvert ?
Quelles garanties manquent chez [Nom] ?
Analyse les gaps de couverture pour [Nom]
```

**Ce que vous obtenez :** Tableau garanties souscrites vs recommandées, gaps prioritaires, axes de développement.

---

### 11. 💶 Comparaison Tarifaire Concurrents
**Ce que ça fait :** Compare les offres GEREP avec celles d'un concurrent pour un client donné.

**Exemples de questions :**
```
Compare nos tarifs avec AXA pour [Nom]
Comment on se situe face à Malakoff Humanis pour [Nom] ?
Benchmark tarifaire Generali vs GEREP pour [Nom]
```

**Ce que vous obtenez :** Tableau comparatif, avantages GEREP, arguments à utiliser en rendez-vous.

## 💡 Conseils & Bonnes Pratiques

### Soyez précis sur le nom du client
AC360 cherche dans SharePoint en utilisant le nom exact. Si la recherche échoue, essayez le nom tel qu'il apparaît dans SharePoint.

✅ `"Résume le dossier ALPHA INDUSTRIE SAS"` → Résultat précis  
⚠️ `"Résume le dossier Alpha"` → Peut retourner plusieurs résultats

### Chaînez les questions
Après une réponse, AC360 propose des suggestions de suivi. Utilisez-les pour naviguer naturellement :
```
1. "Résume le dossier BETA"
2. → Cliquez "Prépare mon RDV de renouvellement"
3. → Cliquez "Rédige un mail avec ces informations"
```

### Précisez le concurrent pour la comparaison
```
"Compare nos tarifs avec AG2R La Mondiale pour GAMMA"
```

## ❓ Que faire si AC360 ne trouve pas d'information ?

| Situation | Signification | Action |
|---|---|---|
| "Je n'ai pas trouvé d'information" | Document absent ou inaccessible | Vérifiez que le dossier est dans SharePoint |
| "Documents insuffisants" | Dossier incomplet | Demandez "Quels documents manquent ?" |
| "Aucune donnée de sinistralité" | Pas de bilan sinistres dans SharePoint | Uploadez les bilans annuels |
| "Aucune donnée concurrentielle" | Pas de devis concurrents dans SharePoint | Uploadez les offres reçues |

## 🔒 Confidentialité & Sécurité

- AC360 **ne voit que vos documents** — pas ceux de vos collègues auxquels vous n'avez pas accès
- AC360 **ne modifie jamais** vos documents SharePoint
- AC360 **ne stocke pas** vos conversations au-delà de la session
- AC360 **ne consulte pas Internet** — uniquement vos documents GEREP
- En cas de doute sur une information : **vérifiez toujours la source citée**
