# Plan POC 2 Semaines — Phase 6

## Vue d'Ensemble

Ce plan POC s'étale sur 2 semaines (10 jours) avec une approche itérative simple.

## Semaine 1 — Préparation

### Jour 1 : Cadrage et Validation

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Valider le besoin avec le responsable | Chef de projet | Accord sur le cas d'usage |
| Choisir les 3-5 cas d'usage prioritaires | Équipe | Liste des cas |
| Identifier les 3-5 commerciaux pilotes | Responsable com | Liste des pilotes |
| Valider le calendrier | Responsable | Dates confirmées |

**Objectif** : validation go/no-go du périmètre

### Jour 2 : Selection des Données

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Sélectionner 10-20 dossiers clients | Commercial | Liste des clients |
| Identifier le site SharePoint cible | IT | URL du site |
| Identifier les bibliothèques concernées | IT | Liste des biblis |
| Documenter la structure existante | IT | Inventaire rapide |

**Objectif** : données ciblées pour le POC

### Jour 3 : Vérification Permissions

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Vérifier les droits SharePoint | IT | Matrice Permissions |
| Vérifier les groupes SharePoint | IT | Liste Groupes |
| Vérifier les propriétaires | IT | Proprios identifiés |
| Identifier les documents sensibles | IT | Liste exclusions |

**Objectif** : sécurité vérifiée

### Jour 4 : Nettoyage Léger

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Nettoyer les dossiers prioritaires | Commercial | Dossiers nettoyés |
| Renommer les documents clés | Commercial | Nommage appliqué |
| Séparer les archives si possible | Commercial | Archives créées |
| Documenter la structure | IT | Doc structure |

**Objectif** : lisibilité améliorée

### Jour 5 : Préparation Agent

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Préparer les questions de test | Chef de projet | 20 questions |
| Rédiger le prompt système | Chef de projet | Prompt finalisé |
| Valider avec le responsable | Chef de projet | Prompt validé |

**Objectif** : agent prêt à créer

## Semaine 2 — Construction et Test

### Jour 6 : Création Agent

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Créer l'agent Copilot Studio | Tech Lead | Agent créé |
| Ajouter source SharePoint | Tech Lead | Source configurée |
| Configurer l'authentification | Tech Lead | Auth MS validée |

**Objectif** : agent opérationnel

### Jour 7 : Configuration et Tests Initiaux

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Configurer les instructions | Tech Lead | Instructions |
| Tester questions simples | Tester | 5 premiers tests |
| Ajuster le prompt si besoin | Tech Lead | Ajustements |

**Objectif** : premiers tests validés

### Jour 8 : Tests Approfondis

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Tester avec plusieurs profils | Tester | 10 tests utilisateurs |
| Vérifier les sources citées | Tester | Qualité notée |
| Tester les cas limites | Tester | Cas limites validés |
| Documenter les résultats | Tester | Rapport tests |

**Objectif** : 15+ tests réalisés

### Jour 9 : Publication Pilote

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Publier à un petit groupe pilote | Tech Lead | Agent publié |
| Former les commerciaux pilotes | Chef de projet | Formation faite |
| Collecter les premiers retours | Pilotes | Retours notés |

**Objectif** : agent en production limitée

### Jour 10 : Démo et Décision

| Tâche | Responsable | Livrable |
|-------|------------|---------|
| Préparer la démo | Chef de projet | Démo prête |

| Présenter les résultats | Chef de projet | Résultats |
| Décider go/no-go extension | Responsable | Décision |

**Objectif** : décision go/no-go
