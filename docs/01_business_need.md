# Business Need — Phase Complémentaire

## Detailed Problem Statement

### Current Situation

Les équipes commerciales de l'entreprise travaillent avec des dossiers clients stockés dans SharePoint. Chaque dossier contient typiquement :

- Contrats (cadres, avenants)
- Propositions commerciales
- Comptes rendus de réunions
- Documents administratifs (factures, KC)
- Échanges importants
- Pièces justificatives

### Pain Points

| Point | Description | Impact |
|-------|-------------|--------|
| Temps de recherche | 30-60 min par dossier | Perte de productivité |
| Recherche manuelle | Parcourir les dossiers un par un | Frustration |
| Informations oubliées | Points clés manqués en réunion | Risque commercial |
| Préparation fastidieuse | Synthèse manuelle avant chaque RDV | Temps excessif |
| Difficulté à retrouver |Documents perdus dans les archives | Délai |

### Users Affected

- **Commerciaux terrain** : 3-5 pilotes identifiés
- **Assistants commerciaux** : Support quotidien
- **Managers commerciaux** : Visibilité sur l'activité

---

## Business Opportunity

### Value Proposition

Créer un agent conversationnel "Assistant Client 360" via Copilot Studio qui permet aux commerciaux de :

1. **Poser des questions** en langage naturel
2. **Obtenir des réponses** synthétisées instantanément
3. **Citer les sources** documents utilisés
4. **Préparer les rendez-vous** en un clin d'œil

### Expected Benefits

| Benefit | Metric | Target |
|---------|--------|--------|
| Gain de temps | Temps préparation | -50% à -80% |
| Productivité | Réponses utiles | >80% |
| Satisfaction | Note commerciaux | >4/5 |
| Sécurité | Fuite documentaire | 0 |

### Competitive Situation

- **Sans solution** : Recherche manuelle, temps long
- **Avec solution** : Réponse instantanée, synthétisée
- **Différenciateur** : Simplicité, sécurité SharePoint, rapidité

---

##fit in Corporate Strategy

Ce POC s'inscrit dans la stratégie Microsoft 365 de l'entreprise :

- Utilisation de Copilot Studio
- Exploitation de SharePoint existant
- Pas de nouveau composant
- Valeur métier rapide
